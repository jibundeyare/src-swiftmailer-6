# Swiftmailer 6

Cette projet montre comment utiliser Swiftmailer pour envoyer des mails
en utilisant directement un serveur SMTP ou un service de
transfert de mail.

## Install

    git clone https://github.com/jibundeyare/src-swiftmailer-6
    composer install
